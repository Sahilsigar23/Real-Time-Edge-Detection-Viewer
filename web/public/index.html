About This Viewer

This TypeScript-based web viewer demonstrates the integration between the Android Edge Detector app and web technologies. It displays processed frames with real-time statistics.

Features

  • Real-time frame statistics display (FPS, resolution, processing time)
  • Base64 image rendering support
  • Modular TypeScript architecture with type safety
  • Clean DOM manipulation
  • Responsive design

Technology Stack

  • TypeScript for type-safe development
  • Modern ES6+ JavaScript features
  • HTML5 Canvas (via img element)
  • CSS3 for styling

Note: In production, frames would be streamed from the Android app via WebSocket or HTTP endpoint. This demo uses a static sample frame.
